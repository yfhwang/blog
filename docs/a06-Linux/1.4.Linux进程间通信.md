## 管道

### 原理

- 伪文件（内核缓冲区）
- 有两个文件描述符(r/w)

### pipe函数

```c
int pipe(int pipefd[2]);		
# 返回值 --- 成功：0；失败：-1，设置errno
```

### 获取管道缓冲区大小

命令：ulimit -a

函数： fpathconf

## FIFO（有名管道）

- 应用于**非血缘关系**进程间通信

- 不能反复读取

### 创建

```
命令：mkfifo
函数：mkfifo
	参数：
		name
		mode：8进制
	返回值：
		成功：0
		失败：-1，设置errno
```
