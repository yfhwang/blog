## 数据存储

MySQL：持久化存储

Redis：访问速度快，用作缓存

方案一：先写Redis，同时异步写入MySQL

```c++
{
  write_to_redis();
  write_to_mysql();
}
```



方案二：直接写入MySQL，使用MySQL将数据同步到Redis

```c
{
  write_to_mysql();
  mysql_sync_redis();
}
```
